#Favorite Music web Demo

# README #
Favorite Music은 스트리밍 음악 콘텐츠 Web, App입니다.

### System ###
* 윈도우 8.1, 윈도우 서버 2012 R2
* Maria DB
* Java 기반 (Spring, mybatis, android)
* javascript, jquery, ajax, bootstrap

### 주요기능 ###
Web [Admin] - 참조
* 음원 콘텐츠 관리
* 음원 저작 관리
* 음원 정산 관리
* 유저 관리

Web [User] - 참조
* 음악 듣기
* 차트
* 라디오

App - 비공개
* Music Radio 기능 (Free)
* 국내 방송 배경음악 토대로 듣기 (Daum API) - 라이센스 Daumkakao
* Chart 
* SNS - Facebook, 트윗터
* 친구추가
